Spark-Hadoop Docker Image

A lightweight Docker image providing Apache Spark with Hadoop libraries and a ready-to-use PySpark environment for local data processing.

Apache Spark Hadoop PySpark Java 17 Docker

Overview

This container bundles Apache Spark built with Hadoop support and runs on Java 17, making it ideal for experimenting with Spark, learning distributed data processing, and testing Spark jobs locally without setting up a full cluster.

The container launches directly into a PySpark shell by default, so you can start writing Spark code immediately.

Pull the Image

docker pull rgkj/spark-hadoop:latest

Run PySpark (Default)

Running the container without arguments launches an interactive PySpark shell. 

docker run -it rgkj/spark-hadoop

Example inside PySpark:

spark.range(10).show()

Access Bash (Root Shell)

If you want full shell access inside the container as the root user, run: 

docker run -it rgkj/spark-hadoop /bin/bash

From here you can manually start Spark, inspect files, or install extra tools.

Run Spark Shell (Scala)

To use the Scala Spark shell instead of PySpark: 

docker run -it rgkj/spark-hadoop spark-shell

Exposing Spark UI

Spark's web UI is available on port 4040. To access it from your browser: 

docker run -it -p 4040:4040 rgkj/spark-hadoop
